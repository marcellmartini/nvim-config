# Plugins - Focus

## Overview

Distraction-free writing and coding modes to help maintain focus.

## Plugins

### zen-mode

- **Repository**: folke/zen-mode.nvim
- **Purpose**: Distraction-free coding environment

#### Keymaps

| Keymap | Mode | Description |
|--------|------|-------------|
| `<leader>z` | n | Toggle Zen Mode |

#### Configuration

- Twilight integration disabled (can be toggled separately)

#### Docs

- https://github.com/folke/zen-mode.nvim

---

### twilight

- **Repository**: folke/twilight.nvim
- **Purpose**: Dims inactive portions of code for better focus

#### Keymaps

| Keymap | Mode | Description |
|--------|------|-------------|
| `<leader>T` | n | Toggle Twilight |

#### Docs

- https://github.com/folke/twilight.nvim
